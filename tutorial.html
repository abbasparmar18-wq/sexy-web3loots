SexyChat — Tutorial

This page explains how to use the SexyChat landing page and the APK download.

How to download the APK

  1. Click the "Download APK" button on the homepage.
  2. The APK will download from GitHub Releases. If your browser warns, allow download.
  3. Install the APK on your Android device. You may need to allow "Install from unknown sources".

How the site is hosted

This site is a static GitHub Pages site. Files in the repository's main branch are served.

Contact

If you want changes, upload new files to the repo or contact the site maintainer.
